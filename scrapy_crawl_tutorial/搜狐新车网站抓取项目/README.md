## Scrapy Crawl

使用scrapy 爬取搜狐新车网站项目示例

## Requirments

- Python2.7
- Scrapy (建议安装Anaconda2.7自带)

## Usage

进入项目的根目录，执行下列命令启动spider:
```
cd .\car_sohu
scrapy crawl sohunewcar

```
